# AutoConnect Buea 🚗🏍️

AutoConnect is a centralized, web-based platform designed to connect stranded vehicle and motorcycle owners with professional mechanics within the Buea municipality in real-time. 

Whether it is a breakdown in Molyko, Great Soppo, or around the Mile 17 corridor, AutoConnect helps users find nearby help quickly, turning a stressful roadside breakdown into an organized, transparent, and efficient digital marketplace.

---

## 🚀 Key Features

### For Vehicle & Motorcycle Owners
* **Proximity Mapping:** Find mechanics based on how close they are to your breakdown location.
* **Specialty Filtering:** Easily filter for specific technicians like Vulcanizers, Auto Electricians, or Engine Specialists.
* **Fault Logging:** Create a request ticket with a brief description and upload photos of the breakdown.
* **Trust & Feedback:** View ratings and leave honest reviews after your vehicle gets fixed.

### For Professional Mechanics
* **Job Dashboard:** Track incoming local job requests and view total earnings history.
* **Availability Toggle:** A simple switch to turn status between "Active" (looking for work) and "Off Duty".
* **Portfolio Profiles:** Showcase specialized equipment or past successful repairs to attract more customers.

---

## 🛠️ Tech Stack

This project is built using modern, lightweight, and open-source industry standards:
* **Frontend UI:** React.js (for a clean, fast, and responsive user experience)
* **Backend & Database:** Supabase / PostgreSQL (for secure authentication, reliable database tracking, and image storage)
* **Maps & Tracking:** Geolocation Services (to calculate distances between drivers and mechanics)

---

## 📋 System Architecture & Flow

The system is developed using the **V-Model Development Lifecycle** to ensure high quality and fewer bugs. The logic is separated into three tiers:
1. **Presentation Layer:** The React web app that users interact with.
2. **Application Layer:** Handles the location matching engine and job broadcasting logic.
3. **Data Layer:** Keeps user profiles, job logs, and ratings securely saved in the database.
